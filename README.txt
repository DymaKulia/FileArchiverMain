
Project has source code, main jar file Archiver-0.0.1-all.jar, 
example of file configuration which is needed for Archiver-0.0.1-all.jar.

Also there is “start check.bat” file which allows to start Archiver in checking mode. 
To use “start check.bat” you need to place it and Archiver-0.0.1-all.jar to same folder and run bat file. 

Archiver-0.0.1-all has two main work modes. Execution  mode and check configuration mode.

To start execution mode you need to enter next: 
java –jar pathToJar\Archiver-0.0.1-all.jar pathToFileConfig

To check configuration mode you need to enter next:  
java –jar pathToJar\Archiver-0.0.1-all.jar pathToFileConfig check
